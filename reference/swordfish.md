# North Atlantic Swordfish dataset

An S4 object containing catch and index time series for North Atlantic
swordfish.

## Usage

``` r
swordfish
```

## Format

An object of class
[MSEtool::Data](https://msetool.openmse.com/reference/Data-class.html).

## Source

ASPIC Software at https://www.mhprager.com/aspic.html

## Examples

``` r
data(swordfish)
```
