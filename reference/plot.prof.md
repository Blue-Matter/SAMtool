# Plot profile object

Generates a profile plot generated by
[profile](https://samtool.openmse.com/reference/profile.md). If a
two-parameter profile is performed, then a contour plot of the
likelihood surface is returned.

## Usage

``` r
# S4 method for class 'prof,missing'
plot(x, contour_levels = 20, ...)
```

## Arguments

- x:

  An object of class
  [prof](https://samtool.openmse.com/reference/prof-class.md) returned
  by [profile](https://samtool.openmse.com/reference/profile.md).

- contour_levels:

  Integer, passed to `nlevels` argument of
  [contour](https://rdrr.io/r/graphics/contour.html).

- ...:

  Miscellaneous. Not used.

## Value

A likelihood profile plot, either a one-dimensional line plot or a
two-dimensional contour plot.
